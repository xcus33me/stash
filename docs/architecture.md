# Stash

Сервис обмена файлами внутри комнат.

# Описание проекта

[product](docs/product.md)
 
# Стек
- Go 1.26, chi (routing)
- PostgreSQL
- MinIO - S3-хранилище

## Хранение файлов

Пишем в MinIO-s3

### На будущее

Файлы пишутся в два места одновременно: локальный диск для быстрой отдачи и MinIO как резервная копия. При сохранении считается SHA256 - он хранится в БД и проверяется cron-скруббингом раз в сутки.

## QR-инвайты

Токен - UUID, хранится в Redis, TTL 5 min.
